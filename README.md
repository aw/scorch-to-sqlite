# Script to convert a Scorch DB (CSV) to SQLite

Get [Scorch here](https://github.com/trapexit/scorch)

# Usage

```
./scorch-to-sqlite.sh /path/to/scorch.db
```

Outputs `/path/to/scorch.db.sqlite`

# Requirements

* gzip
* sqlite3
